## 📝 Query Templates

### 1. Strategy Requests
```
Template:
"Provide a comprehensive [TOPIC] strategy for [GOAL] including 
[ASPECT 1], [ASPECT 2], [ASPECT 3], and [ASPECT 4]"

Example:
"Provide a comprehensive social media strategy for brand awareness 
including platform selection, content types, posting schedule, and 
engagement tactics"
```

### 2. How-To Guides
```
Template:
"Explain everything about [TOPIC] including [ASPECT 1], [ASPECT 2], 
and [ASPECT 3] with examples"

Example:
"Explain everything about email marketing automation including 
segmentation, drip campaigns, and performance tracking with examples"
```

### 3. Analysis Requests
```
Template:
"Analyze [TOPIC] and provide detailed recommendations for [GOAL]"

Example:
"Analyze my campaign performance and provide detailed recommendations 
for improving ROI and conversion rates"
```

### 4. Information Requests
```
Template:
"Tell me everything about [SUBJECT] including [DETAIL 1], 
[DETAIL 2], and [DETAIL 3]"

Example:
"Tell me everything about Google Ads optimization including bidding 
strategies, quality score, and ad extensions"
```

### 5. Comparison Requests
```
Template:
"Compare [OPTION A] vs [OPTION B] with detailed pros, cons, and 
recommendations for [USE CASE]"

Example:
"Compare Instagram Ads vs Facebook Ads with detailed pros, cons, 
and recommendations for e-commerce businesses"
```

## 🎨 Understanding Response Format

### Response Structure

A typical comprehensive response includes:

```
**Main Header** 📊

**Section 1: [Topic]**
Brief introduction or overview paragraph.

**Subsection 1.1: [Subtopic]**
• Bullet point 1
• Bullet point 2
• Bullet point 3

**Subsection 1.2: [Subtopic]**
1. Numbered step 1
2. Numbered step 2
3. Numbered step 3

**Section 2: [Topic]**
More detailed information with examples.

📍 Location-specific info
📞 Contact information
⏰ Time-related details
🌐 Website/online resources
```

### Formatting Elements

**Bold Text:** Important terms, headers, key concepts
```
**Marketing Strategy** → Bold, prominent
```

**Bullet Points:** Lists, features, benefits
```
• Feature 1
• Feature 2
• Feature 3
```

**Numbered Lists:** Steps, sequences, priorities
```
1. First step
2. Second step
3. Third step
```

**Emojis:** Visual markers for specific info
```
📍 Locations
📞 Phone numbers
⏰ Hours/timing
🌐 Websites
📊 Statistics
💰 Pricing
✅ Recommendations
```

## 💡 Pro Tips

### 1. Layer Your Questions
Start broad, then go specific:
```
1st Query: "Overview of social media marketing"
2nd Query: "Detailed Instagram strategy from that overview"
3rd Query: "Specific tactics for Instagram Reels engagement"
```

### 2. Reference Previous Context
The chatbot remembers conversation:
```
You: "Tell me about Instagram ads"
AI: [Provides detailed response]
You: "Now give me specific examples for e-commerce"
AI: [Builds on previous context]
```

### 3. Request Specific Formats
Ask for the structure you want:
```
"Provide this information as:
- A step-by-step guide
- A comparison table
- A priority-ranked list
- A timeline
- Pros and cons"
```

### 4. Combine Multiple Topics
```
"Compare Instagram Ads vs Facebook Ads AND provide budget 
recommendations for a $10,000 monthly spend AND suggest the best 
content types for each platform"
```

### 5. Ask for Examples
```
"Explain [TOPIC] with 3-5 real-world examples"
"Show me sample [CONTENT TYPE] for [USE CASE]"
"Give me templates for [TASK]"
```
